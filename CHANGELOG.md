# Changelog

## 1.2.0 (2026-07-11)

### New Features
- 다국어 지원: 기기 언어 자동 감지 + 설정 화면에서 수동 전환 (한국어·영어·일본어·중국어 간체/번체·스페인어·프랑스어·독일어·포르투갈어·러시아어·이탈리아어·인도네시아어·베트남어)
- 날짜 표기 로케일별 자동 현지화

## 1.1.0 (2026-06-06)

### New Features
- 루틴별 알림 켜기/끄기 (홈 목록 종 아이콘 빠른 토글 + 편집 화면 스위치)
- 슬롯 기본 시간 외에 루틴별 커스텀 알림 시각 직접 지정
- 목록에 적용 중인 알림 시각 표시

## 1.0.2 (2026-04-15)

### Bug Fixes
- 한국 시간 기준 자정에 메인 화면 날짜가 갱신되지 않던 문제 수정 (UTC → 로컬 시간)
- 위젯이 날짜가 바뀌어도 전날 루틴 상태를 보여주던 문제 수정

### Improvements
- 알림 시간 조정: 아침 9시, 오후 3시, 저녁 8시, 언제든 오후 7시
- 루틴 추가/수정 진입 시 키보드 자동 팝업 제거
- 앱 포그라운드 복귀 시 날짜 자동 새로고침
